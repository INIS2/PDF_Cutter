# PDF_Cutter

GUI 기반 PDF 분할 도구입니다. 시작/종료 페이지 또는 페이지 수 기준으로 구간을 정의하고, 여러 PDF로 분할 저장합니다.

## Features
- 인풋 PDF / 아웃풋 폴더 선택
- 표 형태의 분할 정의: 파일명 / 시작 페이지 / 종료 페이지 / 페이지 수
- 행 추가 버튼으로 분할 구간 확장
- 페이지 오프셋 보정 지원
- 파일명에 페이지 범위 자동 추가 옵션
- End Page / Page Count 모드 개별 전환(행 단위)
- 입력 파일명 기반 자동 파일명 채움
- 한/영 UI 토글

## Requirements
- Python 3
- PyPDF2

```bash
pip install PyPDF2
```

## Run
```bash
python PDF_Cutter.py
```

## Notes
- 시작 페이지는 필수입니다.
- End Page와 Page Count는 둘 중 하나만 사용합니다.
- Start만 입력한 경우: 다음 행의 Start - 1을 End로 자동 계산합니다.
- 마지막 행에서 End/Page Count가 없으면 문서 끝까지 분할합니다.
- `.gitignore`에 PDF/이미지/캐시 파일이 포함되어 있습니다.
